# 🖼️ 自动全景图片拼接工具

一个简洁易用的图片拼接软件，支持自动全景图生成。使用 OpenCV 进行高精度的图片对齐和融合。

## ✨ 功能特点

- ✅ **自动全景拼接** - 利用 OpenCV 自动检测和对齐图片
- ✅ **友好的 GUI 界面** - 基于 tkinter，无需命令行操作
- ✅ **实时预览** - 拼接完成后即时显示结果
- ✅ **进度显示** - 清晰的进度条和日志信息
- ✅ **灵活保存** - 支持 JPG、PNG 等多种格式
- ✅ **独立可执行文件** - 打包成 .exe，无需 Python 环境

## 🚀 快速开始

### 方式一：直接运行 EXE（推荐）

1. 下载 `dist/ImageStitcher.exe`
2. 双击运行即可使用，无需任何依赖

### 方式二：Python 直接运行

#### 环境要求
- Python 3.7+
- pip

#### 安装依赖
```bash
pip install -r requirements.txt
