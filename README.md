# 🖼️ 自动全景图片拼接工具

一个简洁易用的图片拼接软件，支持自动全景图生成。使用 OpenCV 进行高精度的图片对齐和融合。

## ✨ 功能特点

- ✅ **自动全景拼接** - 利用 OpenCV 自动检测和对齐图片
- ✅ **友好的 GUI 界面** - 基于 tkinter，无需命令行操作
- ✅ **实时预览** - 拼接完成后即时显示结果
- ✅ **进度显示** - 清晰的进度条和日志信息
- ✅ **灵活保存** - 支持 JPG、PNG 等多种格式
- ✅ **独立可执行文件** - 打包成 .exe，无需 Python 环境

## 🚀 快速开始

### 方式一：直接运行 EXE（推荐）

1. 下载 `dist/ImageStitcher.exe`
2. 双击运行即可使用，无需任何依赖

### 方式二：Python 直接运行

#### 环境要求
- Python 3.7+
- pip

#### 安装依赖
```bash
pip install -r requirements.txt
```

#### 运行应用
```bash
python main.py
```

### 方式三：自己打包成 EXE

#### 安装完整依赖（包括 PyInstaller）
```bash
pip install -r requirements.txt
```

#### 生成 EXE
```bash
python build.py
```

生成的 .exe 文件将在 `dist/` 目录中。

## 📖 使用说明

### 基本步骤

1. **选择图片**
   - 点击 "📁 选择图片" 按钮
   - 选择要拼接的图片（至少 2 张）
   - 建议按拍摄顺序排列

2. **开始拼接**
   - 点击 "🔗 开始拼接" 按钮
   - 等待拼接过程完成（可能需要几秒钟）
   - 在预览窗口查看结果

3. **保存结果**
   - 点击 "💾 保存结果" 按钮
   - 选择保存位置和格式
   - 拼接完成的全景图将被保存

### 使用技巧

#### ✅ 拼接成功的条件

1. **图片重叠** - 相邻图片需要 30-50% 的重叠区域
2. **图片清晰** - 避免模糊或过度曝光的图片
3. **拍摄方式** - 建议使用同一设备拍摄
4. **排列顺序** - 按照从左到右或从上到下的顺序排列

#### ❌ 拼接失败的原因和解决方案

| 错误 | 原因 | 解决方案 |
|------|------|--------|
| 需要更多图片或重叠不足 | 图片之间重叠太少 | 增加图片重叠区域，至少 30% 以上 |
| 无法估计单应矩阵 | 图片特征不匹配 | 使用同一设备拍摄，避免大幅度旋转 |
| 相机参数调整失败 | 拍摄角度差异过大 | 使用更均匀的拍摄角度 |

## 🛠️ 依赖库

- **opencv-python** - 图像处理和全景拼接
- **Pillow** - 图片显示和处理
- **PyInstaller** - 打包成 EXE（仅打包时需要）

## ⚙️ 系统要求

- **操作系统** - Windows 7+（其他系统也支持，但主要针对 Windows）
- **内存** - 至少 2GB（处理大图片建议 4GB+）
- **磁盘** - 至少 100MB 可用空间

## 🎯 高级用法

### 自定义打包设置

编辑 `build.py` 中的参数：

```python
args = [
    'main.py',
    '--name=ImageStitcher',  # 改为你想要的程序名称
    '--onefile',
    '--windowed',
    '--icon=icon.ico',  # 提供自己的图标
    '-y',
]
```

### 添加自定义图标

1. 准备一个 `icon.ico` 文件（256x256 或更大）
2. 放在项目根目录
3. 运行 `python build.py` 即可自动使用

## 🐛 故障排除

### 问题：打包后 .exe 无法运行

**解决方案：**
```bash
# 重新打包
python build.py

# 或清除缓存后重新打包
rmdir /s build
rmdir /s dist
del *.spec
python build.py
```

### 问题：拼接时内存不足

**解决方案：**
- 压缩输入图片大小
- 关闭其他应用释放内存
- 使用更小的图片进行处理

### 问题：无法读取某些格式的图片

**解决方案：**
OpenCV 主要支持：JPG, PNG, BMP, TIFF
- 将图片转换为支持的格式
- 尝试重新保存图片
